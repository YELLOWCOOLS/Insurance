Duoshuo

多说评论系统。

请大家使用的时候把 data-ds-short-name="amazeui" 中的 amazeui 换成自己在多说注册 的网站 ID，不然你们的评论都跑到 Amaze UI 这边来了。
本组件由多说提供服务，更多细节参加官方文档。

目录

  • 组件演示
    • default（默认）
  • 使用方法
  • API

组件演示

default（默认）

<div data-am-widget="duoshuo" class="am-duoshuo am-duoshuo-default" data-ds-short-name="amazeui">
  <div class="ds-thread"></div>
</div>

使用方法

设置多说域名即可，其他参数可选。

API 

{
  "id": "",
  "className": "",
  "theme": "",
  "options": {
    "shortName": "",
  },
  "content": {
    "threadKey": "",
    "title": "",
    "url": ""
  }
}

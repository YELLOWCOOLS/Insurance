单元格状态

表示表格状态的 class 添加到 tr 整行整行，添加到 td 高亮单元格。

  • .am-active 激活；
  • .am-disabled 禁用；
  • .am-primary 蓝色高亮；
  • .am-success 绿色高亮；
  • .am-warning 橙色高亮；
  • .am-danger 红色高亮。
Class状态描述目标元素
.am-active激活td
.am-active激活tr
.am-disabled禁用td

  • .am-text-nowrap: 禁止文字换行；
